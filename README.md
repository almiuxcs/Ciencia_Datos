## ✅ Entregables

El estudiante debe entregar:

1. Link de su repositorio en GitHub  
2. README principal completo  

---

## 📊 Criterios de evaluación

| Criterio | Puntaje |
|--------|--------|
| Repositorio creado correctamente | 20% |
| Subida de prácticas | 30% |
| Organización del repositorio | 20% |
| README completo y claro | 20% |
| Buenas prácticas (nombres, orden) | 10% |

---

## 💡 Recomendaciones
- Usar nombres claros para archivos  
- Subir avances constantemente (commits)  
- Documentar lo aprendido, no solo el código  
- Mantener el repositorio limpio y ordenado
